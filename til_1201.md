## CLI
----
> CLI(Command Line Interface, 커맨트 라인 인터페이스)는 터미널을 통해 사용자와 컴퓨터가 상호 작용하는 방식을 뜻한다.
>
## 터미널 명령어 정리
| 명령어 | 설명                      |
| ------ | ------------------------- |
| mkdir  | 폴더생성                  |
| touch  | 파일생성                  |
| ls     | 현재폴더의 목록출력       |
| sd     | 다른폴더로 이동           |
| rm     | 파일삭제/폴더삭제(-r옵션) |
